# StudyFlow

StudyFlow is a student productivity platform designed to help students manage assignments, organize study schedules, track goals, and improve academic success.

## Mission

Help students stay organized and productive through simple planning tools and future AI-powered study assistance.

## Version 1 Features

- Assignment Tracker
- Daily Planner
- Weekly Goals
- Progress Dashboard

## Future Features

- AI Study Plan Generator
- AI Quiz Generator
- AI Note Summarizer
- AI Flashcard Creator

Planning & Development
